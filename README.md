# Blossom Background

Cinematic full-screen hero section built with Vite, React, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- React
- React DOM
- Framer Motion
- Lucide React
- Tailwind CSS
- Vite
- TypeScript

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The hero background video URL is configured in `.env` via `VITE_HERO_VIDEO_URL`.
- The favicon is loaded from `images/favicon.png`.
